# Sensitivity Analysis (v1)

## What moves outcomes most

### 1) Retention (biggest driver)
If 90-day retention underperforms, LTV collapses and paid acquisition gets constrained.

### 2) Blended ARPU
A $2/month ARPU move materially changes ARR at scale.

### 3) Reach assumptions (SAM/SOM)
Going from 10k to 30k paying families is a bigger leap than most narrative decks imply; requires repeatable channels and strong onboarding.

### 4) Legal/comms friction
If districts resist alternative contact workflows more than expected, activation speed can drop even if legally permissible.

## Quick sensitivity table

Using 30,000 paying families as anchor:

- ARPU $8/mo => $2.88M ARR
- ARPU $10/mo => $3.60M ARR
- ARPU $12/mo => $4.32M ARR

Using ARPU $10/mo as anchor:

- 10,000 paying families => $1.20M ARR
- 30,000 paying families => $3.60M ARR
- 75,000 paying families => $9.00M ARR

## Recommended pre-seed metric discipline
Report monthly:
1. Activated families (weekly active)
2. 4/8/12-week retention
3. Paid conversion of activated users
4. Net alert utility (actionable alerts / total alerts)
5. Cost to activate by channel

These metrics reduce model uncertainty faster than TAM debate.
